# Make My Day — Legal & Support

Public-facing pages for the Make My Day iOS app, served via GitHub Pages.

## Pages

- **[Support / Help](https://chiragthummar.github.io/makemyday-legal/)** — `index.html`
- **[Privacy Policy](https://chiragthummar.github.io/makemyday-legal/privacy.html)** — `privacy.html`
- **[Terms of Use](https://chiragthummar.github.io/makemyday-legal/terms.html)** — `terms.html`

## URLs (use these in the app and App Store listing)

```
Support URL:        https://chiragthummar.github.io/makemyday-legal/
Privacy Policy URL: https://chiragthummar.github.io/makemyday-legal/privacy.html
Terms of Use URL:   https://chiragthummar.github.io/makemyday-legal/terms.html
```

## Updating

Edit any `*.html` file, commit, push to `main`. GitHub Pages picks up changes automatically within ~30 seconds.
