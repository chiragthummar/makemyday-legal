# Make My Day — Legal & Support

Public-facing pages for the Make My Day iOS app, served via GitHub Pages. Generated and refined via the `app-store-upload-guide` skill.

## Pages

- **[Support / Help](https://chiragthummar.github.io/makemyday-legal/)** — `index.html` (rich landing with FAQ, contact, about)
- **[Support (skill template)](https://chiragthummar.github.io/makemyday-legal/support.html)** — `support.html` (boilerplate)
- **[Privacy Policy](https://chiragthummar.github.io/makemyday-legal/privacy.html)** — `privacy.html`
- **[Terms of Service](https://chiragthummar.github.io/makemyday-legal/terms.html)** — `terms.html`
- **[DMCA Policy](https://chiragthummar.github.io/makemyday-legal/dmca.html)** — `dmca.html`

## Updating

Edit any `*.html` file, commit, push to `main`. GitHub Pages picks up changes automatically within ~30 seconds.
